# EOM

Полноразмерный шаблон главной страницы (как в референсе) для:
- WordPress-темы: `wordpress-theme/eom-template`
- GitHub Pages: `index.html`
- локального превью: `preview.html`

## Быстрый просмотр

```bash
python3 -m http.server 8080
```

Откройте `http://localhost:8080/` или `http://localhost:8080/preview.html`.

## WordPress

1. Скопируйте `wordpress-theme/eom-template` в `wp-content/themes/`.
2. Активируйте тему **EOM Landing Template**.
3. Назначьте главную страницу сайта.
